open bitcoin for excel 2.0.03

An open-source python program that provides detailed wallet information about prices, transactions, and blockchain confirmations more directly into excel. It was born from lack of existing libraries.

By default openbtc does not guard against billion laughs xml attacks. To guard against these attacks install defusedxml. To contribute to openbtc, please hack hack hack!
